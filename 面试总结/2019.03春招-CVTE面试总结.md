就像吃饭，三个人排队吃饭，然后开两个窗口，一开始 fun1 和 fun2 去吃饭，然后 fun2 吃完了，这时候 fun3 就上去吃了

下面这是代码，好像是这样的...

```javascript
function queue(list, count) {
  var index = count

  var nextFun = function() {
    if (!list[index]) {
      list[index](nextFun)
    }
    index++
  }

  for (let i = 0; i < count; i++) {
    list[i](nextFun)
  }
}
```
